## IMPORTANT NOTES

1. **Commands are ENGLISH-only** - UI responses are bilingual (fa/en)
2. **Role hierarchy:** SUDO > OWNER > ADMIN > VIP > MEMBER
3. **VIP immunity:** VIP users cannot be auto-punished or banned without removing VIP first
4. **Permissions:** Each admin can have custom permissions via upanel
5. **Panel access:** Most features also accessible via /panel (glass panel UI)
